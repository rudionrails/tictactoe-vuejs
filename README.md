# TicTacToe with Vue.js

## Installation

### Prerequisites

Ensure you have the following software (globally) installed on your system:

* [node.js](http://nodejs.org/) ^6.0.0
* [yarn](https://yarnpkg.com/en/docs/install) ^0.16.0

### Installation

```sh
$ yarn install
```

### Start the development server

```sh
$ yarn dev
```

### Run the test suite

```sh
# run unit tests
$ yarn unit

# run e2e tests
$ yarn e2e

# run all tests
$ yarn test
```

## Deployment

```sh
$ yarn build
```
